# ELAGENT 项目文件说明

## 核心系统（`elagent/`）

| 文件 | 用途 |
|---|---|
| `elagent/main.py` | FastAPI 应用入口，启动服务 |
| `elagent/config.py` | 系统配置（路径、LLM开关、权重阈值等） |
| `elagent/__init__.py` | 包版本号 |
| `elagent/api/routes.py` | **核心管道**：API路由 + 实体链接主流程 `_enhanced_link()` |
| `elagent/api/schemas.py` | Pydantic 请求/响应模型 |
| `elagent/core/knowledge_base.py` | 知识库加载：实体索引、别名检索 |
| `elagent/core/disambiguator.py` | **规则消歧器**：5信号加权 + domain_score(2+3gram) + 长度惩罚 |
| `elagent/core/bm25_index.py` | BM25 全文检索索引 |
| `elagent/core/llm_disambiguator.py` | LLM 消歧（OpenRouter API，gap<0.01触发，持久化缓存） |
| `elagent/core/coref_resolver.py` | 共指消解：代词/指代词回链 + 评测 |
| `elagent/core/nil_detector.py` | NIL 检测器（多信号判定，待接入管道） |
| `elagent/core/trace_logger.py` | 全链路可追溯日志 |
| `elagent/core/vector_index.py` | FAISS 向量索引（未使用） |
| `elagent/core/bert_disambiguator.py` | BERT Bi-Encoder 消歧（已废弃，管道已移除） |
| `elagent/models/mention.py` | Mention 数据类 |
| `elagent/models/entity.py` | Entity / Candidate 数据类 |
| `elagent/models/result.py` | LinkResult / TraceLog 数据类 |

## 数据文件（`Dataset/`、`data/`）

| 文件 | 用途 |
|---|---|
| `Dataset/knowledge_base_merged.json` | **知识库**：6,483 个实体 |
| `Dataset/aliases_merged.json` | 别名 → 实体映射 |
| `Dataset/llm_extracted_merged.json` | 1,239 篇标注文章（实体链接评测用） |
| `Dataset/llm_extracted_with_coref.json` | 合并文件（实体mention + 指代mention，用于共指评测） |
| `Dataset/annotations_with_pronouns.json` | 含指代词标注的文章 |
| `Dataset/NIL.json` | NIL 检测评测数据集（2,527 条 NIL + 212 条非NIL） |
| `data/prior_probabilities.json` | 实体先验概率 |
| `data/all_errors.json` | 评测后自动保存的全部错误（去重） |
| `data/eval_report_full.json` | 实体链接评测报告 |
| `data/eval_report_nil.json` | NIL 检测评测报告 |
| `data/llm_cache.json` | LLM 消歧缓存（持久化，跨运行复用） |

## 评测脚本（`scripts/`）

| 文件 | 用途 | 用法 |
|---|---|---|
| `evaluate_full.py` | **实体链接评测**（链接+消歧+别名） | 主评测 |
| `evaluate_nil.py` | **NIL 检测评测**（准确率、漏检分析） | NIL 评测 |
| `evaluate_coref.py` | **共指消解评测**（代词回链准确率） | 共指评测 |
| `evaluate_all.py` | **全功能评测**（实体链接+共指，按需） | `--no-coref` 跳过共指 |
| `analyze_errors.py` | 错误分类分析 | 🔧 工具 |
| `analyze_gap_types.py` | 消歧分数格局分析 | 🔧 工具 |
| `rebuild_aliases_from_kb.py` | **从 KB 重建别名文件**（修改 KB 后运行） | 数据维护 |
| `clean_knowledge_base.py` / `clean_knowledge_base_v2.py` | KB 清洗 | 🧹 数据清洗 |
| `fix_aliases.py` / `fix_aliases_v2.py` / `fix_aliases_symmetric.py` | 别名修复 | 🧹 数据清洗 |
| `fix_annotations.py` / `fix_annotations_chinese.py` | 标注修复 | 🧹 数据清洗 |
| `fix_kb_chinese_names.py` / `fix_dataset.py` | KB/数据集修复 | 🧹 数据清洗 |
| `merge_entities.py` / `merge_place_names.py` / `merge_tokyo_olympics.py` / `execute_merge.py` | 实体合并 | 🧹 数据清洗 |
| `analyze_merge_candidates.py` | 合并候选分析 | 🧹 数据清洗 |
| `llm_auto_judge.py` / `llm_entity_judge.py` / `llm_merge_judge.py` | LLM 辅助判定 | 🧹 数据清洗 |
| `review_entity_pairs.py` / `rebuild_aliases.py` | 实体对审核/别名重建 | 🧹 数据清洗 |
| `generate_report.py` / `generate_research_report.py` / `generate_testset.py` | 报告/测试集生成 | 📊 报告 |
